# Teologando

Teologando é um projeto voltado a estudos teológicos e apologética cristã, com foco em materiais que apoiem o estudo bíblico e histórico.

**Visão**
- Oferecer conteúdo claro e bem fundamentado para quem busca entender questões doutrinárias e históricas à luz das Escrituras.

**Resumo técnico**
- Implementado com frameworks modernos para web (Next.js + TypeScript) e estilização responsiva.

**Como rodar localmente**
1. Instale dependências:

```
npm install
```

2. Rodar em modo de desenvolvimento:

```
npm run dev
```

3. Gerar build de produção:

```
npm run build
```

4. Iniciar servidor de produção (após `build`):

```
npm start
```
